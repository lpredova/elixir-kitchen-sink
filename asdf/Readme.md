# asdf



https://github.com/asdf-vm/asdf


### Installing elixir

```sh
asdf plugin-add elixir
```

```sh
asdf list-all elixir
```

```sh
asdf list elixir
```

```sh
asdf local elixir 1.8.2
```

### Installing erlang

```sh
asdf plugin-add erlang
```

```sh
asdf list-all erlang
```

```sh
asdf list erlang
```

```sh
asdf global erlang 22.2.8
```
